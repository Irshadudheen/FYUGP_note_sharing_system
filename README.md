# FYUGP Note Sharing System

A web application for **students and teachers to upload, share, and download academic notes** efficiently. This platform promotes collaborative learning by making notes accessible to all registered users.

## Features

- **Student Uploads:** Upload notes in PDF, DOC, or image formats.  
- **Teacher Contributions:** Teachers can add study materials.  
- **Search & Filter:** Browse notes by subject, course, or keywords.  
- **Ratings & Feedback:** Optional feature to rate high-quality notes.  
- **Secure Access:** User authentication ensures safe usage.  

## Technology Stack

- **Frontend:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)  
- **Backend:** [Node.js](https://nodejs.org/) + [Express](https://expressjs.com/)  
- **Database:** [MongoDB](https://www.mongodb.com/)  
- **File Storage:** [Cloudinary](https://cloudinary.com/) for notes and media  
- **Authentication:** JWT-based secure login system
